## ğŸ¯ How to Play

1. A random word is selected (represented by blank spaces)
2. Guess letters by clicking the on-screen keyboard or pressing keys
3. Correct guesses reveal the letter(s) in the word
4. Wrong guesses add parts to the hangman drawing
5. Win by guessing the word before the hangman is complete
6. Use hints strategically when needed (limited to 2 per game)
